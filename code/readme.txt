The code 'SmartReader.py' is to read seven-segment numbers. It requires a set of images with fixed camera position. In the code, users need to define the region of interest by dragging a rectangle on the image. Then, the code will use that information for the rest of the image set. There are some functions that the code offers: 1) adjusting brightness 2) sharpening 3) deleting dots 4) plotting and 5) calculating operation time. This program can be instrumental in increasing the accuracy and the speed with which typical people type.
